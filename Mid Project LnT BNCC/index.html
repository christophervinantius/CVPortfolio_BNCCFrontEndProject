ABOUT ME


CHRISTOPHER VINANTIUS

Front-End Web Developer



Saya adalah mahasiswa BINUS University Kemanggisan (Binusian 2024) jurusan Teknik Informatika (School of Computer Science). Selain itu, saya menjadi member organisasi BNCC (Bina Nusantara Computer Club) dan mengikuti kelas LnT (Learning and Training) Web Design.

Saya mengambil kelas Web Design karena saya tertarik untuk memperdalam pengetahuan serta memperkaya pengalaman mengenai pemrograman front-end dari sebuah website.

Sebelumnya saya sudah mempelajari dasar-dasar pemrograman front-end sejak lulus Sekolah Menengah Atas, sekaligus mempelajari dasar-dasar pemrograman dengan bahasa C untuk persiapan perkuliahan.

Awalnya hanya sekadar 'iseng', namun perlahan saya merasa bahwa saya memang tertarik seputar desain website. Saya tidak hanya ingin sekadar mendesain, tetapi saya juga ingin menuangkan ide-ide desain yang saya miliki dalam bentuk kodingan dengan berbagai bahasa pemrograman yang umumnya digunakan.

Pekerjaan impian saya di masa depan adalah menjadi seorang front-end web developer, karena menurut saya, pekerjaan ini akan dibutuhkan dalam jumlah banyak di masa depan seiring dengan perkembangan teknologi pengembangan website.
